# Tableau Prep Builder

Este repositório reúne práticas desenvolvidas no **Tableau Prep Builder**, com foco em preparação, limpeza, transformação e organização de dados para análise.

Os arquivos disponíveis são fluxos empacotados no formato `.tflx`, que podem conter o fluxo de preparação, metadados e bases de dados utilizadas nas atividades.

## Conteúdos Abordados

- Importação de bases de dados
- Limpeza e padronização de campos
- Renomeação de colunas
- Tratamento de dados para análise
- União de tabelas
- Organização de fluxos de preparação
- Criação de saídas prontas para visualização no Tableau

## Descrição dos Arquivos

### Movies.tflx

Fluxo de preparação de dados relacionado a uma base de filmes.

O pacote contém uma planilha de origem e um fluxo de tratamento no Tableau Prep Builder.

Principais pontos trabalhados:

- Conexão com arquivo Excel
- Leitura de base de filmes
- Organização de campos
- Preparação dos dados para análise posterior

### P2 - Eletronicos.tflx

Fluxo de preparação de dados relacionado a uma base de eletrônicos.

O pacote contém uma planilha de origem e etapas de transformação no Tableau Prep Builder.

Principais pontos trabalhados:

- União de tabelas
- Padronização de origem dos dados
- Renomeação de campos
- Preparação de uma base consolidada

## Como Abrir os Arquivos

Para visualizar ou editar os fluxos:

1. Instale o **Tableau Prep Builder**.
2. Baixe ou clone este repositório.
3. Acesse a pasta `Práticas`.
4. Abra o arquivo `.tflx` desejado no Tableau Prep Builder.

## Observação

Arquivos `.tflx` são fluxos empacotados do Tableau Prep Builder. Eles podem incluir tanto o fluxo de preparação quanto os dados utilizados na prática.

Alguns arquivos podem representar atividades preliminares ou versões intermediárias dos exercícios.

## Objetivo do Repositório

Este repositório tem como objetivo organizar práticas de preparação de dados no Tableau Prep Builder, demonstrando etapas de tratamento, padronização e estruturação de bases para análise e visualização.
